What and Why

This plugin shows 4 basic Projections and can be used with OpenCPN as is, or for programming support for various projections.

Installation

The OpenCPN Manual has general information in Plugin Download, Install and Enable for installing this plugin.

How to set the Projection

Right click anywhere on the chart and pick the Projection from the list available.
